Yummy User Interface Project

Overview

This project involved conducting research in the form of user interviews, competitive analysis, use cases, scenarios etc. to improve the user experience for the Yummi application. Yummi is a food journaling application. My team and I enhanced the user’s posting and filtering experience. The filter feature will allow users to browse through their and friends’ photos in an easier and more efficient manner just as the posting feature will also be efficient.
